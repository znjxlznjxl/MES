# 📱 MES: Mobile Element Selector

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![Mobile Friendly](https://img.shields.io/badge/Mobile-Optimized-orange?style=for-the-badge)
![License: MIT](https://img.shields.io/badge/License-MIT-green.svg?style=for-the-badge)

개발자 도구(F12)를 사용할 수 없는 **모바일 브라우저 환경**에서 웹 페이지의 요소를 직접 터치하여 선택하고 정보를 확인할 수 있게 해주는 유저스크립트입니다.

---

## 🚀 주요 기능
* **터치 요소 선택:** 모바일 화면에서 원하는 부분을 터치하여 해당 요소의 태그, 클래스, ID 등을 확인합니다.
* **경량화:** 모바일 브라우저에서도 끊김 없이 작동하도록 최적화되었습니다.
* **디버깅 보조:** 모바일 UI 레이아웃을 수정하거나 스크립트 제작 시 타겟 요소를 찾을 때 유용합니다.

---

## 🛠️ 사용 방법

1.  스크립트를 설치한 후, 분석을 원하는 웹 페이지에 접속합니다.
2.  스크립트가 활성화되면 화면의 요소를 터치하여 정보를 확인합니다.
3.  (추가적인 UI가 있다면 해당 인터페이스를 통해 제어합니다.)

---

## 📲 권장 환경
* **Android:** Kiwi Browser, Lemur Browser (Tampermonkey 지원 브라우저)
* **iOS:** Safari (Userscripts 확장 프로그램 설치 시)
